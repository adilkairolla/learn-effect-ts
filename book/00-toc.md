- **"I want to write a backend service":** Part I → chapters 22–28 (`@effect/platform`, `@effect/sql` core + drivers, `@effect/rpc`) → chapters 33–34 (observability, schedule) → Part III.
- **"I want to write a CLI tool":** Part I → chapters 19–21 (`@effect/cli`, `@effect/printer`, `@effect/printer-ansi`) → Part III.
- **"I want to author a library":** Part I → chapters 11, 46–60 (constructors, Part III). Part II is reference, not required reading.
- **"I want to build AI-powered applications":** Part I → chapters 22–23 (platform + Node.js), 31–32 (`@effect/ai` + providers) → Part III.
- **"I want distributed systems / durable execution":** Part I → chapters 25–30 (SQL, RPC, workflow, cluster) → Part III.
